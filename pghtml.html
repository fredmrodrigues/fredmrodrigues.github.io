HTML: Hypertext Markup Language

O HTML pode ser definido como o bloco de construção principal da web. Através deste arquivo, é possível criar a estrutura do conteúdo de uma página ou aplicação web. Existem outras tecnologias que, geralmente, são utilizadas para desenvolver a aparência da página, como o CSS ou a funcionalidade/comportamento da página, como o JavaScript.

"Hipertexto" faz referência aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites distintos. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um avegador da Web. A marcação HTML inclui "elementos" especiais denominadas tags.

Um elemento HTML é separado de outro texto em um documento com as tags, que consistem no nome do elemento entre "<" e ">". O nome de um elemento dentro de uma tag é insensível a maiúsculas e minúsculas. Isto é, pode ser escrito em maiúsculas, minúsculas ou ambas, embora seja de boa prática usar minúsculas.

Este módulo lhe permite começar a criar as partes visíveis dos formulários, ou seja, as partes que os usuários interagem.

    Apresenta links para seções com conteúdo explicando como usar HTML para resolver problemas muito comuns, quando se cria uma página web, como trabalhar com títulos, agregar imagens ou vídeos, colocar ênfase em conteúdo ou criar um formulário básico.
